An Android native application that delivers entertainment and real-time content updates for upcoming, trending, and popular/top-rated content based on user feedback.

Built using Android/Kotlin, it serves as a case study for Stateflow and upcoming technologies like Jetpack Compose, Hilt, animations, and more.
