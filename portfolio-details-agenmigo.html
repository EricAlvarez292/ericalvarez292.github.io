AgenMigo

Project information

  • Category: Mobile and Backend Application
  • Project Type: Company Project
  • Company: Migo Entertainment Inc.
  • Project URL: AgenMigo

An incentive program platform driven by the Sales Team that facilitates incentive program management, agent account/organizational management, and sales dashboard, boosting company sales by 25% and providing extra income to over 3,000 external agents.

Built with Android/Kotlin for the app and Angular, NodeJS, ExpressJS, PostgreSQL, and AWS for full-stack web development.
